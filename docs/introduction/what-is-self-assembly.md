# 自己集合とは何か

自己集合（self-assembly）とは、分子や分子集合体が、外から「組み立てる操作」をしなくても、**分子同士の相互作用に従って、自発的に秩序だった構造を作る現象**である。
ポイントは「勝手に凝集する」のではなく、**「決まった形に集まる」**ことにある。

たとえば、水に油が混ざらずに分かれるのも「自発的な構造化」の一種なのだけれど、自己集合ではさらに一歩進んで、ミセル、二重膜、繊維、らせん、リングなどの**形がある集合体**を作れる。

---

## 自己集合が起こる理由：自由エネルギーが負になるから

自己集合に限らないが、自発的に変化が起こる条件は **自由エネルギー変化が負になること（Δ*G* < 0）** である。  
熱力学的には、このような式で表される：

Δ*G* = Δ*H* − *T*Δ*S*

ここで、Δ*G*はギブズエネルギー変化、Δ*H*はエンタルピー変化、Tは温度、Δ*S*はエントロピー変化である。

Δ*H*とΔ*S*はイメージしづらいが、直観的には次のように捉えるとわかりやすい。

- **Δ*H*（エンタルピー変化）**：相互作用ができてエネルギー的に得をする分（得をすると負になる（Δ*H* < 0））  
- **Δ*S*（エントロピー変化）**：「系の乱雑さ（どれだけ無秩序に散らばっているか）」に対応する分（散らばると正になる（Δ*S* > 0））

つまり、自己集合の観点で、Δ*G*を負になるようにするためには、以下の条件が必要なことがわかる：

- **Δ*H* < 0**：分子同士が相互作用し、安定化する。
- **Δ*S* > 0**：分子がより分散して散らばる。

ここで、Δ*S*に注目していただきたい。
自己集合すると、必ず **エントロピーは減少する（Δ*S* < 0）** はずである。
ばらばらの分子が整列するのであれば、乱雑さはなくなってしまうからだ。
なので、エントロピーの観点では、自己集合は必ず不利になる。（例外：疎水相互作用）

つまり、自己集合が進むためには、分子同士が相互作用してエンタルピーを稼ぐしか方法がない。
そのため、超分子ポリマーの設計は、Δ*H*を稼ぐことのできる相互作用ユニットを組み込む必要がある。

代表例：
- 水素結合
- π–π 相互作用
- 静電相互作用
- 金属配位 など

## 集合体の例

- **ミセル**：界面活性剤が球状に集合  
- **二重膜**：脂質が膜構造を作る  
- **繊維（ナノファイバー）**：一次元に伸びる集合体（＝超分子ポリマー）  
- **らせん**：キラルな情報が増幅され、ねじれ構造が現れる。キラルな分子が集まると、ねじれの向きが一方向に定まる。  
- **リング**：ファイバーの末端が偶然つながるとリングに！末端がないので分子がはがれ落ちにくく超安定！

これらの構造体は一種類に固定されるとは限らない。
条件次第で共存したり、時間とともに別の形へ変わったりすることもある。
