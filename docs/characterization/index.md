# 測定方法まとめ：どの手法で何が分かる？

超分子ポリマーは「分子が集まる」だけでは終わらない。  
**どんな形で集まり、どれくらい秩序があり、どれくらい動的に入れ替わるか**で性質が決まる。  
そのため、測定手法は「強い手法を1つ」ではなく、**役割の違う手法を組み合わせる**のが基本になる。

このページでは、各手法で何が分かるかを“一発で”整理する。

## 目的別に選ぶ

- **集まった？（モノマー vs 集合体）**  
  → UV/Vis / CD / FL / NMR

- **どう並んだ？（秩序・ねじれ・配列）**  
  → CD / UV/Vis / XRD

- **サイズは？形は？（ナノ構造のスケール）**  
  → SAXS/SANS/DLS / 顕微鏡（AFM/TEM/SEM）

- **どんな相互作用？（水素結合・配位など）**  
  → IR / NMR / UV/Vis（配位変化）

- **動的？遅い？（キネティクス・履歴・可逆性）**  
  → 時間・温度・濃度依存UV/Vis・CD・FL

## 一覧表：手法 → 得意分野

| 手法 | 何を“直接”見ている？ | 長所 | わかること |
|---|---|---|---|
| UV/Vis | 電子遷移（吸収） | 速い・手軽・滴定/温度/時間に強い | 集合の進行、配列変化の気配、重合曲線、ヒステリシス |
| CD | 左右円偏光の吸収差 | 秩序・ねじれに敏感 | 超分子キラリティー、配列の切替、秩序化の進行 |
| FL | 発光（消光/増強） | 微量でも効く・環境に敏感 | 会合/解離、エネルギー移動、励起子・凝集の指標 |
| IR | 振動（N–H, C=Oなど） | 相互作用の“痕跡”が見える | 水素結合の形成/変化、溶媒和、官能基状態 |
| NMR | 化学シフト/線幅/拡散など | 分子レベルの情報 | 会合状態、配位、交換の速さ、場合によりサイズ感（DOSY） |
| SAXS/SANS | ナノ構造の散乱 | 溶液中の形・サイズに強い | 半径、断面、長さの推定、ミセル/繊維/チューブの判別補助 |
| DLS | 動的散乱（揺らぎ） | 速い・粒径の目安 | サイズの目安、凝集の検出（ただし形の解像度は低い） |
| XRD | 結晶/秩序構造の回折 | 高秩序に強い | 分子間距離、積層周期、結晶化・高配向の確認 |
| AFM | 表面の形状 | 形が“見える” | 繊維/リング、長さ分布、太さの比較（乾燥条件に注意） |
| TEM/SEM | 電子顕微鏡像 | 高分解能で形が見える | 形状、内部構造（TEM）、ネットワーク、粒子観察 |

## 重要な注意

### 1) 手法によって「見えるもの」が違うことがある
同じ試料でも、手法ごとに見えている情報が違う。  
例：  
- UV/Vis：電子状態の変化  
- SAXS：サイズと形（平均像）  
- 顕微鏡：実像（ただし前処理の影響あり）

だから、結果が一致しないのではなく、**見ているものが違う**ことが多い。

### 2) “溶液中”と“乾燥後”は別物になりうる
AFM/TEMは構造体を直接見られる点では強いが、試料を基板に乗せる時点で状態が変わることがある。  
溶液中の状態を押さえたいなら、散乱や分光の情報とセットで読む。

### 3) 散乱法（SAXS/DLS）は「形」を当てるにはモデルがいる
散乱は強いが、解釈はモデル依存になりやすい。  
最初は「サイズ感」「形の候補」を出す道具として使うのが安全。適用したモデルがあっているか顕微鏡像と比較する必要がある。

## おすすめの“最短”セット（迷ったらこれ）

- **まず状態変化を追う**：UV/Vis（＋FL）  
- **配列・秩序を確認**：CD（キラルな分子なら）  
- **相互作用の手がかり**：IR（＋必要ならNMR）  
- **形・サイズの裏取り**：SAXS/DLS と AFM/TEM
