# 超分子ポリマー入門

**生体のすごさは、分子が勝手に集まって“形”を作り、機能を生み出すことにある。**
**超分子ポリマーは、その仕組みを人工分子で再現し、条件しだいで形を変えたり、伸び縮みしたり、つながったり離れたりする“動く材料”を扱う考え方だ。**
**このサイトでは、自己集合の仕組みから測定法までを、できるだけ直感で分かるように解説する。**
